# 💱 Currency Converter App

A simple and interactive Currency Converter built with **React.js**.  
It uses live exchange rates from an API and allows fast currency conversion with a clean UI.

---

## 🚀 Features

- 🔄 Real-time currency conversion
- 🌍 Supports multiple currencies
- 🔁 Swap from/to currencies instantly
- 📡 Live API data using custom React hook
- 🎯 Clean and responsive UI
- ⚡ Fast and simple user experience

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- Tailwind CSS
- Currency Exchange API
